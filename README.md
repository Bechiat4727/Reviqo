# Plateforme Web de Présentation de Magasins

**Reviqo** est une plateforme web simple permettant de présenter différents magasins et leurs informations.  
Le site est principalement statique, avec un backend léger utilisant Flask pour le rendu des pages.

## Fonctionnalités principales

- Pages web statiques présentant les magasins et leurs informations.
- Interface simple et claire avec une navigation facile.
- Backend Flask minimal pour servir les pages HTML.

## Technologies utilisées

- HTML et CSS pour la structure et le style des pages.
- JavaScript pour de petites interactions côté client (animations ou boutons simples).
- Python avec Flask pour le rendu des pages côté serveur.

## Installation et lancement

1. Cloner le dépôt :
```bash
git clone https://github.com/Bechiat4727/Reviqo.git
```

2. Installer les dépendances Python (Flask):
```bash
pip install -r requirements.txt
```

3. Lancer le serveur Flask :
```bash
python login_backend.py
```

4. Ouvrir le navigateur à l’adresse :
```bash
http://127.0.0.1:5000
```

## >⚠️ Le site est principalement statique. Les boutons et éléments interactifs sont limités.

## Objectifs du projet

 - Mettre en pratique les compétences en développement front-end (HTML, CSS, JavaScript).

 - Apprendre à structurer une application web simple avec Flask.

 - Créer un site web clair et présentable, même avec des fonctionnalités limitées.

## Résultats et acquis

 - Développement d’une plateforme web fonctionnelle, simple et bien présentée.

 - Renforcement des compétences en HTML, CSS, JavaScript et Flask.

 - Apprentissage des bonnes pratiques pour organiser un projet web statique avec un backend léger.
